# Puppeteer Boilerplate

Puppeteer end-to-end test automation boilerplate, Using Commonjs, Cucumber, Chai and Should.

## Getting Started

    	1. git clone git@github.com:e2e-boilerplate/puppeteer-commonjs-cucumber-chai-should.git
    	2. cd puppeteer-commonjs-cucumber-chai-should
    	3. npm install
    	4. npm run test
